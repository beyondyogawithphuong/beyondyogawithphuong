# Modern Minimalist Redesign

## New Design System

### Colors
- **Primary**: #212121 (Neutral-900 - text/buttons)
- **Secondary**: #757575 (Neutral-600 - subtext)
- **Tertiary**: #E0E0E0 (Neutral-300 - dividers)
- **Background**: #FAFAFA (Neutral-50 for sections)
- **White**: #FFFFFF (main background)

### Typography
- **Headings**: Georgia serif, font-weight 400, letter-spacing -0.02em
- **Body**: System sans-serif, line-height 1.6
- **Size hierarchy**: 3rem → 2rem → 1.5rem

### Spacing
- Large sections: py-24 md:py-32
- Medium sections: py-16 md:py-20
- Breathing room between elements

### Components
- **Buttons**: Simple black/white with hover states
- **Cards**: Minimal border, lots of whitespace
- **Dividers**: Thin gray lines (1px, #E0E0E0)
- **Accents**: Minimal, focus on typography

## Key Changes
- Remove all colored backgrounds
- Use white/gray/black only
- Let your logo/photo be the main visual
- Focus on typography and spacing
- Remove rounded corners (use 0px)
- Minimal borders
- More breathing room
